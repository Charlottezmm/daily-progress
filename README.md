## 这是什么

> 实时进度看板 · 2026-05-07 → 2026-05-31
> 主线：Karpathy "Zero to Hero" + 开源 Agent 工具 v0.1

每天自动定位「今天是 Day X」，显示当日任务 + 自检问题 + 当周 milestone。勾选状态和文字编辑都保存在浏览器 localStorage（不跨设备同步；可通过左侧 sidebar 的导出/导入 JSON 在多设备间手动迁移）。

## 配套

- 本页：每天扫一眼今天该做什么；标题/任务/自检/milestone **点击即可在线编辑**
- 私有 Notion：每天晚上写当日反思
- 每天 5:00 自动简报：Claude scheduled task
